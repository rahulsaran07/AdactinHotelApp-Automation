 # Adactin Hotel App - Automation Testing

 This project automates the functional testing of the Adactin Hotel web application
 using Selenium WebDriver and Java. It validates the critical features such as login 
 and hotel booking flows through automated test scripts.

 # Tools used
  - Java (JDK 17)
  - Selenium WebDriver
  - Maven (for dependency management)
  - JUnit (for test execution)
  - Eclipse IDE
  - Git & GitHub (version control)
 
 # Project Structure
  - src/main/java             # Core project files
  - src/test/java             # Test scripts
  - pom.xml                   # Maven configurations
  - TestCaseDocument.xlsx     # Manual test case documentation
  - Readme.md                 # Project overview
  
 # Test scenarios covered
  - Login functionality validation
  - Search hotel with valid data
  - Search hotel with mandatory fields
  - Booking confirmation validation
  
 # Running the test
  - Clone the repository :  git@github.com:rahulsaran07/AdactinHotelApp-Automation.git
  - Open the project in Eclipse or any IDE supporting Maven
  - Update the project dependencies 
  - Run the test 
  
 # Test documentation
  - Detailed manual test cases are included in the "TestCases.xlsx" file located in the root directory.  
    Each test case includes steps, expected results, actual results, and test status.
